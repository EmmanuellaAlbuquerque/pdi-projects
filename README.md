# pdi-projects

## Configuring Python Virtual Environment

### Installing dependencies
```sh
# inside root project folder
~$ sudo -H pip3 install virtualenv
~$ virtualenv pdi-env
~$ . pdi-env/bin/activate
~$ pip3 install -r requirements.txt
```

```sh
# inside src folder
```
